Dette er dokumentationen for, hvordan du bruger elementerne i Design System CSS - en implementation af SDFIs design guide.

Den komplette designguide kan findes på Figma. (Kræver SDFI login.)

Koden og tilhørende dokumentation findes på Github.

Design System CSS er baseret på pico.css. Du kan finde deres dokumentation her.
